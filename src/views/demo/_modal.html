To open the modal, you can use javascript to add the class modal-show as sibling class next to class modal.

Modal

  To make the modal be visible, you need to add the class modal-show as sibling class next to class:modal.
  And you can close the modal by removing the class modal-show.
  <div class="modal">
    <div class="modal__wrapper">
      <div class="modal__header">
        You can adjust the title position by using either the class modal__header--title-left, modal__header--title-center, or modal__header--title-right
        <p class="modal__header--title modal__header--title-center">
          Title
        </p>
      </div>
      <div class="modal__content>
        To open the modal, you can use javascript to add the class modal-show as sibling class next to class modal.
      </div>
      <div class="modal__footer">
        You can adjust the buttons position by using either the class modal__footer--btns-left, modal__footer--btns-center, or modal__footer--btns-right
        <div class="modal__footer--btns modal__footer--btns-right">
          <button class="btn-negative btn-sm" onclick="closeModal()">Close</button>
          <button class="btn-positive btn-sm" onclick="closeModal()">OK</button>
        </div>
      </div>
    </div>
  </div>
