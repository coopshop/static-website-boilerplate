Responsive Navbar

      Applying navbar-sticky allows the navbar to stick to the top.
      <div class="navbar navbar-sticky">
        <div class="nav">
          <ul>
            <li><a href="#">Menu</a></li>
            Adding class active to menu item is to highlight the current/active menu item.
            <li class="active"><a href="#">Active Menu</a></li>
          </ul>
        </div>

        The class nav--responsive is to convert the menu into a responsive navigation menu.
        Resize the browser window to see how the responsive navigation menu works.
        The responsive menu is set to hover menu as default.
        If you want to keep the menu opened, you can add nav--responsive-open as sibling class next to nav--responsive.
        <div class="nav nav--responsive">
          <div class="i-menu"><span></span></div>
          <ul>
            <li><a href="#">Responsive Menu</a></li>
            <li class="active"><a href="#">Active Responsive Menu</a></li>
          </ul>
        </div>
      </div>
