Tabs

    <div class="tabs">
      The 'id' value must match the 'for' value in label, e.g. id="tab1", for="tab1". Otherwise, the tab won't be triggered when clicked.
      <input type="radio" name="tab" id="tab1" data-tab="tabcontent1"><label for="tab1">Tab 1</label>
      The 'data-tab' value must be started with tabcontent as prefix and ended with a number between 1 and 30, e.g. data-tab="tabcontent1", data-tab="tabcontent30"
      <input type="radio" name="tab" id="tab2" data-tab="tabcontent2"><label for="tab2">Tab 2</label>
      To show the tab content, the tab content id must match the corresponding data-tab value.
      <div class="tabcontent" id="tabcontent1">
        Lorem ipsum dolor sit amet,
      </div>
      <div class="tabcontent" id="tabcontent2">
        Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
      </div>
    </div>
