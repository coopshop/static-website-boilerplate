Textarea

    <textarea placeholder="textarea"></textarea>
    You can use the class error to indicate the invalid content.
    <textarea placeholder="textarea with invalid content" class="error"></textarea>
    Use the class resize-v to allow user resizing textarea's height only.
    Use the class resize-h to allow user resizing textarea's width only.
    Use the class resize-none to restrict user to resize textarea.
    <textarea placeholder="textarea (only height can be resized)" class="resize-v"></textarea>
    Use the class filled to form a textarea with color filled.
    <textarea placeholder="textarea in filled style" class="filled"></textarea>
